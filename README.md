## üõ†Ô∏è Portfolio Architecture

This portfolio uses Astro's island architecture with strategic hydration:

- **Astro Pages**: Static pages for optimal performance
- **React Modules**: Interactive sections hydrated with client directives
  - `client:load` - Hero, Header (immediate interactivity)
  - `client:visible` - About, Projects, Contact, Experience (lazy load on scroll)

### Key Features
- Hybrid rendering strategy (static + interactive islands)
- Emotion CSS-in-JS for component styling
- Path aliases for clean imports
- Vite compression plugin for optimized builds
- WebP images for optimal loading
- Responsive design with mobile-first approach
