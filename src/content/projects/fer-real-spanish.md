## FerRealSpanish — Language Learning Platform

Full-stack Spanish learning platform serving 500+ waitlist users. Implemented row-level security (RLS) for multi-tenant student data, typed REST APIs for cohort management, and automated email campaigns. Deployed via Vercel with GitHub Actions CI/CD.

### What it does
- Onboards 500+ waitlist users with Supabase auth + RLS
- Cohort-based course management with typed REST APIs
- Automated email sequences via Resend + Apache Airflow DAGs
- Continuous deploy on every push via GitHub Actions
