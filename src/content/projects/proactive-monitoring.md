## Proactive Monitoring System — RPA Operations Dashboard

Built 4 real-time monitoring dashboards (React, Next.js, TypeScript, TailwindCSS) serving 120+ operations users, reducing incident response time by 35% through centralized KPI visibility.

### What it does
- Real-time KPI visibility across 4 RPA pipelines
- Alert routing with on-call rotation integration
- Historical drilldowns into incident root-cause patterns
- Operations user-facing dashboard for 120+ users

### Stack
React · Next.js · TypeScript · TailwindCSS frontend, Django REST + PostgreSQL backend.
